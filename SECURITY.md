# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Multibot, please report it responsibly. **Do not open a public issue.**

**Email**: [security@codance.ai](mailto:security@codance.ai)

### What to Include

- A clear description of the vulnerability
- Steps to reproduce the issue
- Impact assessment (what an attacker could achieve)
- Any suggested fixes, if applicable

### Response Timeline

- **Acknowledgment**: within 48 hours
- **Status update**: within 7 days
- **Fix release**: as soon as a patch is ready, coordinated with the reporter

We appreciate your help in keeping Multibot and its users safe.
